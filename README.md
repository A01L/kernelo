# 🌌 Kernelo Framework

**Kernelo** — это высокопроизводительный Node.js фреймворк, спроектированный с акцентом на модульность, прозрачность и строгую изоляцию системного ядра от бизнес-логики.

---

## 🏛 Архитектурный регламент

Система построена по принципу разделения ответственности. Структура проекта минимизирует риски при масштабировании и упрощает аудит кода.

### Иерархия директорий:
* **`kernel/`** — Неизменяемое ядро системы.
    * **`routers/`** — Реестр и логика распределения трафика.
    * **`modules/`** — Нативные системные модули Kernelo.
    * **`configs/`** — Параметры среды и конфигурации окружения.
    * **`main.js`** — Центральный процессор движка (классы, методы, инициализация).
* **`containers/`** — Изолированная среда для разработки прикладного функционала.
* **`node_modules/`** — Внешние зависимости и библиотеки.
* **`index.js`** — Точка входа в систему (Entry Point), инициализирующая ядро.

---

## 🛠 Ключевые возможности

### 🛰 Kernelo Watchtower (Built-in Web Interface)
В состав ядра включен проприетарный модуль мониторинга, предоставляющий полный контроль над инфраструктурой в реальном времени:
* **Project Monitor:** Глобальное отслеживание состояния всех процессов.
* **Container Status:** Анализ производительности отдельных контейнеров.
* **Traffic Inspector:** Глубокая инспекция роутов и входящих/исходящих потоков.
* **System Metrics:** Мониторинг нагрузки на CPU, RAM и файловую систему.
